SECTION ONE

HTML & CSS

An introduction to HTML and CSS where you'll master the art of responsive web designs. An approach to creating web pages called responsive design uses flexible layouts, flexible images, and media queries in cascading style sheets. The objective of responsive design is to create web pages that recognize the size and orientation of the visitor's screen and adjust the layout accordingly.

Duration: 3 weeks

SECTION TWO

DESIGN LOOPS

In this section you will complete at least 10 projects using HTML and CSS. Here you will apply the skills acquired in section 1. Duration: 2 weeks

SECTION THREE

JAVASCRIPT

Both basic and advance language features of JavaScript are highlighted in this section. Basic syntax, closures, functions, DOM, arrays, objects, loops, and many other topics are covered.

At least three javaScript projects must be finished by the end of this section.

Duration: 4 weeks

SECTION FOUR

Putting Everything Together

In this section you will complete a project using HTML, CSS and JAVASCRIPT. You will learn how to fix bug in your code.

Duration: 3 weeks
